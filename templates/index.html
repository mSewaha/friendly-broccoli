friendly broccoli

Just a raspberry pi 3 connected to an Enviro-pHat which has weather and motion sensors on it, that write to a firebase database using a python script that is executed hourly. A python web framework called Django was used for the web-app’s back-end and UIKit was used to make the website look the way it looks(front-end).
